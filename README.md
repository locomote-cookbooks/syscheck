# syscheck (LWRP)

## Description

This LWRP provides http://github.com/locochris/syscheck bash based scripts to check system status.
(Which is in turn built on top of the marvelous http://github.com/shpec/shpec)

## Usage

See our *actual* recipes found in the http://github.com/locomote-cookbooks/loco-syscheck cookbook that use this LWRP.
